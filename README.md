# GymDash

A React dashboard for tracking gym sessions, exercises, and personal records.

Built with React, TypeScript, and Vite.

## Features

- Log workouts and individual sets
- Track personal records per exercise
- View session history at a glance
- Fast, client-side — no backend required

## Tech stack

- React 18
- TypeScript
- Vite
- CSS Modules

## Getting started

**Prerequisites:** Node.js 18+

```bash
# Clone the repo
git clone https://github.com/Sintonice/Gymdash.git
cd Gymdash

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Available scripts

| Command         | Description                        |
|-----------------|------------------------------------|
| `npm run dev`   | Start local development server     |
| `npm run build` | Build for production (outputs `dist/`) |
| `npm run preview` | Preview the production build locally |
| `npm run lint`  | Run ESLint                         |
